# Mehrak Bot

Mehrak Bot is a Discord Bot for HoYoverse games, supporting all 4 major titles from HoYoverse and providing different utilities across the board

Mehrak integrates with [Discord User Apps](https://support-apps.discord.com/hc/en-us/articles/26593412574359-How-to-Use-Apps), allowing users to access Mehrak commands in other servers, without requiring a server invite

Mehrak also features a simple CRUD maintainer dashboard, aimed to allow contributions from non-technical maintainers with minimal code changes
